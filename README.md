## Project Overview

A cross-device mall directory web application serving **28 shops** across entertainment, beauty, food, and lifestyle categories — built entirely in vanilla HTML, CSS, and JavaScript with **zero external framework dependencies**.

### Features

- Browse shops by floor
- View ratings and reviews for individual shops
- Filter shops by category
- Identify vegetarian-friendly food options
- Responsive, adaptive layouts across devices

### Pages (8 screens)

- Home
- Shop Listing
- Individual Shop Detail
- Category Filter
- Login
- Register
- About Us
- Contact Us

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla — no frameworks)
